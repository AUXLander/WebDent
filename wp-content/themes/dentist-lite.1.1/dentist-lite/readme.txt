== Description ==
Dentist Lite is a free dental WordPress theme. It is perfect for medical ,dental, spa, yoga, health care, hospital and 

any oher flat business. It is user friendly customizer options. it is Compatible with WooCommerce, Nextgen gallery 

,Contact Form 7 and many WordPress popular plugins.
